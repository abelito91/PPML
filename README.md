# PPML

Le but de ce projet était d'appliquer un methode économetrique (PPLM Pseudo Poisson Likelihood Methods) qui prend en compte les zeros dans une grande quantités de données (+10000).

Pour cela, je suis allé chercher des facteurs économiques qui pouvaient influencer et les regrouper par proximité pour réaliser une regression linéaire. 
Parmis ces facteurs par pays nous avons : 
- le taux d'homicides 
- l'espérance de vie 
- l'indice de corruption

Vous retrouverez donc ici les differentes données csv  et le script R pour la regression.
